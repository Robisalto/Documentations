Lister les vols ou il pleut à l'arrivée ou au départ.

Les vols avec de la pluie à l'arrivée devront sortir avant les autres.

GET /kibana_sample_data_flights/_search
{
  "query": {
    "multi_match": {
      "query": "Rain",
      "fields": ["OriginWeather", "DestWeather^3"]
    }
  }
}

Lister les vols partant d'Allemagne et à destination de France

GET /kibana_sample_data_flights/_search
{
  "query": {
    "bool": {
      "must": [

        { "match": { "OriginCountry": "DE" }}
      ]
    }
